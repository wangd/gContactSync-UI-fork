Source Code

The gcontactsync source code can be accessed through one of the following ways:

  1. View the source using the CVS web interface
  2. Check out the source using anonymous CVS with the following commands (password is guest): 
    cvs -d :pserver:guest@mozdev.org:/cvs login
cvs -z3 -d :pserver:guest@mozdev.org:/cvs co gcontactsync/src2
