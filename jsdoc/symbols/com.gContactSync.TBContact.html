Class Detail
com.gContactSync.TBContact(aContact, aDirectory)
Makes a new TBContact object that has functions to get and set various values for a contact independently of the version of Thunderbird (using com.gContactSync.GAbManager). Optionally takes the parent directory and is able to update the card in that directory.
Parameters:
aContact
{nsIAbCard} A Thunderbird contact.
aDirectory
{AddressBook} The parent directory. Optional.
